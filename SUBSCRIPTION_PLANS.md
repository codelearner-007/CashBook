## Plan Rules

### Free
- Data stored on device only — no cloud backup or sync
- Can view Customers & Suppliers and Categories but cannot add, edit, or delete
- Can view Reports but cannot download (PDF/Excel) or share
- Data is lost if app is uninstalled or phone is wiped

### Pro
- Full cloud sync — data accessible across multiple devices
- Full access to all features except Guest Access
- 7-day backup history — can restore data from any point in the last 7 days

### Business
- Full cloud sync — data accessible across multiple devices
- 30-day backup history — can restore data from any point in the last 30 days
- Guest Access — invite up to 10 guests per account
  - Owner assigns each guest a permission level: View only, Edit, or Full access
  - Guests can access the owner's cloud books based on their assigned permission
  - Guests' own data remains local unless they purchase their own subscription

## Guest Access Permission Levels

| Permission | Can View | Can Add Entries | Can Edit/Delete | Can manage Books/Categories |
|---|---|---|---|---|
| View only | Yes | No | No | No |
| Edit | Yes | Yes | Yes | No |
| Full | Yes | Yes | Yes | Yes |
